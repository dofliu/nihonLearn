# 手動走查清單 — 瀏覽器端

自動化測了邏輯與後端；以下是需在瀏覽器實跑的部分。三個終端：

```bash
# 終端 1：mock VOICEVOX（未裝真 engine 時）
cd sidecar && uvicorn mock_voicevox:app --port 50021
# 終端 2：sidecar
cd sidecar && VOICEVOX_URL=http://127.0.0.1:50021 ENABLE_WHISPER= uvicorn main:app --port 8848
# 終端 3：前端
npm run dev   # http://localhost:5173
```

## かな / ことば（FSRS）
- [ ] 假名翻面自評四鍵可用；秒答的字明顯拉長不再出現、忘了的字稍後重現。
- [ ] 「音→字」測驗需先學過 ≥4 枚才可玩。
- [ ] ことば道場：看日文＋🔊 → 翻面中文，四鍵評分；進度地圖／統計更新。

## 聴く（辨音 + 重音）
- [ ] 辨音道場 5 題，答完慢速對比重播兩詞。
- [ ] 重音道場：高低線圖正確渲染（紅點高、灰點低）；最小對立組（雨/飴、箸/橋）對比清楚；型別測驗可作答。

## 話す（跟讀 + 評分）
- [ ] 語音來源標示正確（VOICEVOX / 瀏覽器）。
- [ ] **whisper 在線**（sidecar 設 `ENABLE_WHISPER=1` 且有 faster-whisper）：麥克風鈕 🎤→⏹ 手動起停，評分後顯示逐拍診斷（綠/紅/黃）。
- [ ] **whisper 未開**：降級瀏覽器語音辨識（自動偵測語尾）。
- [ ] **無麥克風權限**：降級自評三鍵。
- [ ] 「✨ 生成新練習句」→ 選主題生成候選；每句顯示覆蓋率與新詞；超綱句亮紅警示；採用後回「話す」對應層級可見該句。

## 読む（漢字模式）
- [ ] 短文點句切換中文並朗讀；「読了」計入任務。
- [ ] 設定 → 漢字モード ON：詞彙卡正面顯漢字、單字帳顯漢字附假名、短文振り仮名隱藏。OFF 復原。

## 語音設定（VOICEVOX）
- [ ] 設定頁「重新偵測 sidecar」→ 偵測到後出現說話者下拉與試聽。
- [ ] 切換說話者後試聽聲音改變；重整後設定保留。
- [ ] 關閉 sidecar 後重新偵測 → 退回瀏覽器語音，功能不中斷。
